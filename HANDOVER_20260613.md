## PROVISIONAL NUMBERS — REVIEW REGISTER

**Governing principle:** Every hand-set numeric threshold in the strategy is PROVISIONAL — a current best estimate from thin data, NOT a validated constant. A provisional number that is never revisited becomes a fabricated rule (cf. the 5.5% CHOPPY floor that sat for weeks looking authoritative). Review is mandatory, not optional. Long-term goal: these numbers become DATA-DERIVED via the learning loop, not hand-set.

> **⚠️ STANDING RULE — TP/SL COUPLING CHECK (must run before shipping any cap change):**
> TP caps and SL placement are **coupled** through the R:R gate: `blended reward = 0.30×TP1 + 0.70×TP2 ≥ 1.1 × SL distance`. Tightening a TP cap without a matching SL tightens the required SL to pass the gate — and can silently make a whole regime un-enterable. This is the mirror-image of the old 5.5% CHOPPY SL floor (which blocked entry from the SL side); the CHOPPY TP2 at 2.5% did the same from the TP side (8/9 historical CHOPPY trades would have been gate-rejected). **Before shipping any change to a TP or SL cap in any regime: compute `blended reward / realistic SL ≥ 1.1` using the SL distances Claude actually sets for that regime. If it fails, the cap change is not viable.**
