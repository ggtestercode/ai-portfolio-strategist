## 4. KEY FINDINGS / LESSONS (don't relitigate)
- **Three "rules" were never code-enforced** — discovered via audit before building. Rule 7's phantom-order protection (the reason it was written) did not exist at all. Rule 8 had no price guard. TP2 wasn't gated. Building Rule 7 the obvious way (`bybitGetOrders()` for everything) would have produced constant false "missing" alarms — `getOrders()` calls `/v5/order/realtime?orderFilter=Order` which returns regular limit orders ONLY; Bybit position-level TP/SL (set via `/v5/position/trading-stop`) are invisible there and only appear on the position object from `bybitGetPositions()`. The audit caught this before the code was written.
- **Learning loop was broken in ~5 ways** before tonight: dead-code attribution path (190, harmless), orphaned IDs on every regen (~37/72 trades crediting nothing — the real attribution killer), no selectivity (all rules = global win rate), 42% of sl_hit corpus mislabeled (8/19 were ratcheted wins), delete-all regen with no backup. All addressed.
- **Per-rule attribution can only ever validate ~5 rules** (regime-gated 11/14, directional 5/10, partial 9). Universal rules = global rate by nature. **Vetoes (Rule 1, 5) are structurally unmeasurable** — the trades they prevent don't exist. Don't expect W/L to rank these.
- **XMR loss diagnosis**: per-symbol regime opened it correctly, but ADX=46→54 was LAGGING a +40% blowoff that had already reversed (ADX can't tell "trending" from "just finished trending"). SL was placed in noise ($363 vs real swing $352) to clear R:R, then wicked (closed back above $363). NOT a per-symbol-regime failure — an exhaustion-entry + noise-SL failure. Both addressed (blowoff metrics + no-noise-zone SL rule).
- **Per-symbol regime live evidence is only ~2 points**: XMR (lost), ATOM (open). Don't over-trust the feature yet.
- **Win rate is the real lever** — ~40% on recent 10-trade sample at ~1:1 R:R is net negative. Fixes must lift WIN RATE, not just R:R.
- **Recurring pattern this session**: first investigation got the shape right, specifics wrong (ATOM metrics, single-candle blowoff bug, swing_low-never-read, corpus-23-vs-12 trap). Verification before building caught each. Keep doing that.
